# 두루마리 (Durumari) v2 📜

두루마리는 옛 조선시대 서책과 두루마리를 읽는 듯한 고풍스러운 느낌과 현대적인 편의성을 결합한 텍스트 뷰어 앱입니다. 

## ✨ 주요 기능

* **대용량 파일 지원 완벽 최적화**: 기존의 메모리 부족(OOM) 문제를 완벽히 해결하기 위해 SQLite와 React Native Webview Canvas를 결합하여 아무리 큰 텍스트 파일도 버벅임 없이 쾌적하게 읽어옵니다.
* **폴더 단위 동기화 관리**: 여러 폴더를 동시에 관리하고 라이브러리에 탭으로 추가하여 파일들을 분류할 수 있습니다. 
* **아름답고 고풍스러운 UI**: 조선시대 분위기가 물씬 나는 애니메이션 및 한지(Paper) 테마를 통해 우아한 독서 경험을 제공합니다. (라이트/다크 테마도 지원)
* **네비게이션 바 및 상태표시줄 동기화**: 앱의 테마 설정에 맞춰 스마트폰의 시스템 바 색상도 완벽하게 연동됩니다.
* **다양한 페이지 넘김 옵션**: 터치, 스와이프는 물론 볼륨 키를 이용한 페이지 넘김도 지원하며, 책장 넘김 효과 애니메이션 및 진동 피드백도 선택할 수 있습니다.
* **디테일한 정렬 및 라이브러리 기능**: 제목, 파일 일자, 상태, 진행률 등 다양한 기준을 터치하여 목록을 오름차순/내림차순으로 정렬할 수 있으며 설정은 기기에 안전하게 저장됩니다.

## 🛠 기술 스택

* **React Native & Expo (v56)**: 크로스 플랫폼 앱 개발.
* **SQLite (expo-sqlite)**: 대용량 데이터 및 메타데이터, 책갈피, 히스토리 고속 처리.
* **WebView (react-native-webview)**: Canvas 기반의 고성능 텍스트 렌더링.
* **File System**: `expo-file-system`, `expo-document-picker`를 통한 안전하고 직관적인 파일 접근 방식 연동 (SAF).

## 🚀 시작하기

### 필요 환경
* Node.js
* Expo CLI
* Android Studio (안드로이드 빌드 시)

### 설치 및 실행

1. **저장소 클론 및 패키지 설치**
   ```bash
   npm install
   ```

2. **안드로이드 환경에서 실행**
   ```bash
   npx expo run:android
   ```

3. **기본 개발 서버 실행**
   ```bash
   npx expo start
   ```

## ⚙️ 설정 및 맞춤화

설정 화면에서는 다음과 같은 요소를 기호에 맞게 변경할 수 있습니다:
* 글자 크기, 줄 간격, 자간, 굵기 조정
* 폰트 서체 변경 (나눔명조, KoPub 바탕체 등)
* 여백 디테일 조정 (상, 하, 좌, 우)
* 뷰어 조작 방식 (스와이프, 터치, 볼륨 키)
* 페이지 넘김 효과 (없음, 슬라이드, 책장 넘김)
* 피드백 방식 (없음, 진동, 소리)
