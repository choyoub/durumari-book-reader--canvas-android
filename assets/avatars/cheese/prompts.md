# Cheese Cat V1 Prompt Set

These prompts generated the concept sheets under `concepts/`.
All sheets use the same character baseline:

- Cute but realistic-proportioned orange tabby cat
- Cream muzzle, chest, paws, and belly where visible
- Warm brown tabby stripes
- Amber eyes
- Slim kitten-like body
- Expressive ears and tail
- Polished 2D game sprite concept art
- Sprite-friendly clean silhouette with subtle pixel-art inspired edges
- No text, watermark, UI, accessories, extra cats, huge head, or chibi body

## Reference

Create a polished reference sheet with four poses:
side view standing, front view sitting, back view sitting, side view sitting.

## Sit

Create one behavior sheet focused only on sitting.
Rows: idle sitting, tail sway, drowsy sitting, blink, ear twitch,
head turn/look back, front paw tidy, tiny nose bubble while dozing.

## Lie

Create one behavior sheet focused only on lying.
Rows: side lying idle, tail-tip flick, eyes closing and opening,
forepaw stretch, small body roll, belly-up relaxed pose.

## Sleep

Create one behavior sheet focused only on sleeping.
Rows: curled breathing loop, tiny nose bubble, ear twitch,
paw twitch, weak tail movement.

## Walk

Create one behavior sheet focused only on walking.
Rows: slow side-view walk cycle, pause/hesitate, sniff floor,
turn around, short playful run.

## Stretch

Create one behavior sheet focused only on stretching.
Rows: front paws stretch forward, back arch, big yawn, tail rises.

## Groom

Create one behavior sheet focused only on grooming.
Rows: lick front paw, wash face, lick side body, clean ear with paw.

## Play

Create one behavior sheet focused only on gentle play.
Rows: short hop, floor tap, chase tail, sudden freeze and stare.

## Alert

Create one behavior sheet focused only on user-touch reaction.
Rows: startled tiny jump, eyes widen, look back over shoulder, ears perk.

## Drag

Create one behavior sheet focused only on scruff drag.
Rows: scruff hold lifted from above, limp dangling body,
little paws moving/dangling, tail swaying left and right while held.
The hand holds only the loose neck scruff; no distress or injury.

## Fall

Create one behavior sheet focused only on safe falling.
Rows: body curled while falling, paws spread while falling,
tail waving for balance.

## Land

Create one behavior sheet focused only on safe landing.
Rows: crouch on landing, shake body/fur, recover back to sitting.
